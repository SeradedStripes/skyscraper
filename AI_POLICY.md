# Contributing to Skyscraper --- The AI Policy

## Rules

### Can you prove your code works, and do you understand it?

If you can't explain what your change does without an AI tool then don't submit it.
If you can't show us that your code works and runs then it can't exactly be merged can it...

### Showing it works

Every PR needs evidence you ran the code, anything works for this:
- A screen recording
- A screenshot (before/after)
- Pasted test outputs
This can be simple but it should scale with the complexity of the change.

### All AI Usage must be disclosed

If you used any AI tools to help you write code, documentation, or anything else for this PR you must disclose it in the PR description. This is not a problem, we just want to know what was used and how it was used. This helps us know what to expect from the PR and helps us understand the code better.
